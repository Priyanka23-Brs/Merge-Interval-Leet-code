Merge Interval

Find the Merge Intervals for the given array

The merge function sorts the intervals by their start time, initializes a result array next, and then iterates through the sorted intervals. If the end time of the current interval in next is greater than or equal to the start time of the next interval in the original array, it merges the two intervals by taking the maximum of their end times. Otherwise, it adds the next interval to the next array. The intervals are represented as pairs of numbers, where the first number is the start of the interval and the second number is the end of the interval.
